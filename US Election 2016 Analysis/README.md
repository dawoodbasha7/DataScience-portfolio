# US Election 2016 Exploratory Data Analysis

This repository contains Python code for conducting Exploratory Data Analysis (EDA) on the US Election 2016 dataset. The dataset provides insights into various aspects of the 2016 US elections.

## Dataset

The dataset used for this analysis includes information such as voter demographics, candidate preferences, voting patterns, and other relevant variables. It is crucial for understanding the dynamics and outcomes of the 2016 US presidential election.

## Dependencies

To run the code in this repository, you will need the following Python libraries:

- `pandas`: Data manipulation and analysis
- `numpy`: Numerical computing
- `matplotlib`: Data visualization
- `seaborn`: Statistical data visualization
- `opendatasets`: To downloading datasets from online sources like Kaggle using API

All the dependencies are listed in Requirements.txt file

You can install these dependencies using pip:

```bash
pip install pandas numpy matplotlib seaborn opendatasets
```

## File Structure

- `US_Election_2016_EDA.ipynb`: Jupyter Notebook containing the Python code for EDA.
- `Requirements.txt`: The list of dependencies used for the analysis.
- `README.md`: Overview of the repository.

## Usage

1. Clone this repository to your local machine:

```bash
git clone https://github.com/TuringCollegeSubmissions/ddudek-DWWP.4.git
```

2. Navigate to the repository directory:

```bash
cd ./US_Election_2016_Analysis.ipynb
```

3. Run each cell in the notebook to execute the code and generate visualizations.
